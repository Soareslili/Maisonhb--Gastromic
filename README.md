# MaisonHD - Restaurante Premium

Projeto de site institucional para um restaurante de alto padrão, desenvolvido com foco em elegância, experiência do usuário e conversão de clientes.

O objetivo do projeto é apresentar o restaurante de forma sofisticada, destacando seus serviços, ambiente e experiência gastronômica, além de facilitar o contato e reservas online.

---

## Tecnologias Utilizadas

- React
- TypeScript
- Vite
- TailwindCSS
- React Router DOM
- AOS (Animate On Scroll)

---

## Funcionalidades

✔ Navegação entre páginas com React Router  
✔ Layout 100% responsivo (mobile, tablet e desktop)  
✔ Animações suaves com AOS  
✔ Integração visual moderna e elegante  
✔ Estrutura organizada e escalável  
✔ Página exclusiva para reservas  

---

## Estrutura do Projeto

O projeto foi desenvolvido com navegação em múltiplas páginas utilizando React Router DOM, garantindo uma experiência fluida para o usuário.

### Páginas:

- Home
- Menu
- Sobre
- Galeria
- Depoimentos
- Contato
- Reserva

---

## Seções do Projeto

### Hero Section

A primeira seção do site apresenta uma imagem impactante de um prato premium, com overlay escuro e tipografia sofisticada.

**Destaques:**
- Headline chamativa com destaque em cores douradas
- Botões de ação:
  - "Faça sua reserva"
  - "Explore o Menu"
- Design focado em conversão
- Background com imagem de alta qualidade

---

### About (Sobre)

Seção que transmite autoridade, qualidade e experiência do restaurante.

**Destaques:**
- Texto institucional com storytelling
- Destaque para:
  - Ingredientes premium
  - Chef executivo premiado
  - Atendimento personalizado
- Lista de diferenciais com ícones
- Layout dividido (texto + imagem)
- Estilo sofisticado com cores escuras e douradas

---

### Menu

Apresentação dos pratos oferecidos pelo restaurante.

**Funcionalidades:**
- Organização visual dos pratos
- Layout moderno e intuitivo
- Destaque para imagens e descrições

---

### Galeria

Seção visual para apresentar o ambiente e pratos do restaurante.

**Destaques:**
- Imagens de alta qualidade
- Grid responsivo
- Foco na experiência visual do cliente

---

### Depoimentos

Seção com feedbacks de clientes para gerar credibilidade.

**Destaques:**
- Prova social
- Layout moderno
- Destaque para avaliações positivas

---

### Contato

Área para facilitar a comunicação com o cliente.

**Possui:**
- Informações de contato
- Layout claro e acessível
- Foco em conversão

---

### Página de Reserva

Página dedicada para reservas de mesa.

**Destaques:**
- Interface simples e objetiva
- Foco na experiência do usuário
- Estrutura pronta para integração com backend futuramente
